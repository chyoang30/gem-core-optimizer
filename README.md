\# ArkGrid 젬 최적화 도구



LostArk의 스펙업 시스템인 \*\*ArkGrid\*\*에서 젬을 코어에 효율적으로 배치하기 위한 최적화 도구입니다.

브라우저에서 실행되는 간단한 웹 도구로, 젬 목록을 입력하면 세 개의 코어에 가장 효율적인 배치를 계산합니다.



\## 주요 기능

\- CSV 파일을 통한 젬 목록 불러오기

\- 스프레드시트 붙여넣기 지원

\- 젬 클릭으로 코어에 배치

\- 자동 최적화 기능

\- 코어 상태 시각화



\## 사용 방법



1\. 젬 목록을 CSV 업로드 또는 붙여넣기로 입력합니다.

2\. 코어 등급을 설정합니다.

3\. 수동으로 젬을 배치하거나 \*\*자동 최적화\*\*를 실행합니다.

4\. 가장 효율적인 코어 배치를 확인합니다.



\## 기술 스택

\- HTML

\- CSS

\- JavaScript



별도의 서버 없이 브라우저에서 바로 실행됩니다.



\## 목적



ArkGrid 젬 배치를 직접 계산하는 과정이 번거로워 편하게 최적 조합을 찾을 수 있도록 만든 개인용 도구입니다.
